# HireHub

**HireHub** is a modern React application that allows users to find, book, and manage skilled workers for various tasks. Admins or users can add workers, while users can view available workers, filter/search, book them, and view booking history.  

This project demonstrates **React Context for state management, React Router for navigation, and modular component design**.

---

## Features

- Responsive and modern UI using CSS modules  
- Add workers with details (name, profession, hourly rate)  
- List of workers with **search and filter by profession**  
- Edit and delete workers  
- Book workers with total price calculation  
- Booking history to track all bookings  
- Ready for future API integration
